# 💰 Bank Loan Dashboard | Power BI

Developed an interactive **Bank Loan Dashboard** in **Power BI** to analyze loan performance, borrower profiles, and lending trends. The dashboard provides key insights into loan distribution, customer demographics, income variance, home ownership, verification status, and state-wise loan analysis through interactive visualizations and slicers.

### 📊 Key Features

* Total Loan Amount, Average Interest Rate, Customer Count, and Income Variance KPIs
* State-wise Loan Distribution Analysis
* Home Ownership Breakdown (Rent, Mortgage, Own, etc.)
* Loan Trend Analysis by Issue Date
* High-Risk Loan Analysis by Verification Status
* Interactive Filters for State and Grade
* Clean, user-friendly dashboard design for business decision-making

### 🛠️ Tools & Technologies

* **Power BI**
* **Power Query**
* **DAX**
* **Data Modeling**
* **Data Visualization**

### 📈 Business Insights

* Identifies states with the highest loan disbursement.
* Analyzes borrower home ownership patterns.
* Tracks loan issuance trends over time.
* Highlights high-risk loans based on verification status.
* Enables dynamic filtering for deeper analysis.
